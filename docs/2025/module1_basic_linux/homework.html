📝 Linux Homework: Bioinformatics Basics

🔧 Objectives

  • Practice file and directory management
  • Navigate the Linux filesystem
  • View file contents
  • Compress and extract folders

📁 Part 1: Setup

Run the following commands in your terminal:

mkdir bio_hw
cd bio_hw
mkdir raw processed results scripts
touch raw/sample1.fastq raw/sample2.fasta
touch scripts/run.sh scripts/clean.sh

📂 Part 2: File Navigation

Try these commands and observe the output:

pwd
ls
cd raw
ls
cd ..
ls -R

Question: What is the full path to sample1.fastq?

📄 Part 3: File Management

Move, copy, and rename files:

mv raw/sample2.fasta processed/
cp raw/sample1.fastq results/
mv scripts/clean.sh scripts/cleanup.sh

👁️ Part 4: Viewing Files

Try the following:

cat raw/sample1.fastq
head raw/sample1.fastq
file raw/sample1.fastq

Question: What do these commands show for an empty file?

📦 Part 5: Compression

Compress your folder:

zip -r bio_hw.zip bio_hw/
tar -czvf bio_hw.tar.gz bio_hw/

Decompress your archives:

unzip bio_hw.zip
tar -xzvf bio_hw.tar.gz

Format Compress Decompress
