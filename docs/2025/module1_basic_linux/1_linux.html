Operating System

Commonly known as “OS”

An operating system is a software that manages a computer's hardware and software resources, providing common services for computer programs. It acts as an intermediary between users and the computer hardware, facilitating tasks such as memory management, process scheduling, and input/output operations. The OS is the first program that loads when you start your computer, enabling the system to function effectively.

Most common operating systems include:

  • Windows: Windows 10, Windows 8, Windows 7, Windows XP, Windows Vista
  • macOS: Mojave, High Sierra, Sierra
  • Linux: Red Hat, Fedora, SUSE, Ubuntu, openSUSE, Slackware, Gentoo
  • Android OS
  • iOS

UNIX OS

Stable, Multi-Tasking, Multi-User Operating System: UNIX is a robust operating system that supports multiple users and multitasking capabilities, making it ideal for various computing environments.

Origin: Originally developed by AT&T’s Bell Labs in the mid-1960s, UNIX has been foundational in the evolution of modern operating systems.

Open Standard: UNIX is an open standard owned and managed by The Open Group, ensuring consistency and interoperability across different implementations. Learn more at The Open Group.

Variants: UNIX has several variants maintained by different companies, including:

  • MacOS
  • Solaris
  • AIX
  • HP-UX
  • BSD

Linux Kernel: The Linux kernel was derived from UNIX, inheriting many of its design principles and features.

LINUX

Unix-like Operating System Family: Linux is a family of Unix-like operating systems built under the model of free and open-source software development and distribution. It is one of the most prominent examples of open source development and free software; its underlying source code is available for anyone to use, modify, and redistribute freely.

First Released by Linus Torvalds in 1991: While still a student at the University of Helsinki, Linus Torvalds started developing Linux to create a system similar to MINIX, a UNIX operating system. In 1991, he released version 0.02; Version 1.0 of the Linux kernel, the core of the operating system, was released in 1994.

Open-Source: Linux is open-source, meaning it is free to obtain and modify. The source code of Linux may be used, modified, and distributed commercially or non-commercially by anyone under the terms of its respective licenses, such as the GNU General Public License (GPL).

Dominance in Supercomputing: Linux is the predominant operating system for servers and is also used on all of the world's 500 fastest supercomputers. As measured by the TOP500 list, which uses HPL to measure computational power.
