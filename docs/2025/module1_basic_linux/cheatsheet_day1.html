Linux Commands Cheatsheet

Basic Syntax

[command] [options] [arguments]

Example: ls -l /home/user

Filesystem Navigation

Command Description Sample Usage
pwdPrint current working directorypwd
lsList files and directories in current folderls
ls -aList all files, including hidden filesls -a
cd <directory>Change to the directory specified by <directory>cd Documents
cd ..Move up one directory levelcd ..

File Management

Command Description Sample Usage
mkdir <folder>Create a new folder named <folder>mkdir project
touch <file>Create an empty file named <file>touch notes.txt
cp <source> <destination>Copy file from <source> to <destination>cp file1.txt file2.txt
cp -r <source> <destination>Recursively copy a folder from <source> to <destination>cp -r folder1 folder2
mv <source> <destination>Move or rename file from <source> to <destination>mv old.txt new.txt
rm <file>Delete the file named <file>rm unwanted.txt
rm -r <folder>Recursively delete folder named <folder>rm -r old_folder

Viewing Files

Command Description Sample Usage
cat <file>Display the contents of the file <file>cat file.txt
less <file>View the file <file> one screen at a timeless bigfile.txt
head <file>Show the first 10 lines of the file <file>head file.txt
head -n <num> <file>Show the first <num> lines of the file <file>head -n 20 file.txt
tail <file>Show the last 10 lines of the file <file>tail file.txt
tail -n <num> <file>Show the last <num> lines of the file <file>tail -n 20 file.txt
tail -f <file>Follow the live updates of the file <file>tail -f log.txt

Compressing Files

Command Description Sample Usage

zip -r <archive.zip> <folder>Compress a folder into a ZIP archive recursivelyzip -r project.zip project_folder
tar -cvf <file.tar> <folder>Create a tar archive from a foldertar -cvf backup.tar my_folder
tar -czvf <file.tar.gz> <folder>Create a gzip-compressed tar filetar -czvf backup.tar.gz my_folder
tar -cjvf <file.tar.bz2> <folder>Create a bzip2-compressed tar filetar -cjvf backup.tar.bz2 my_folder
gzip <file>Compress a file using gzipgzip data.txt
