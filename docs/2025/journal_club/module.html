The mechanics of Journal Club

At the end of every week, a group of students will present a paper or a software tool

  • The participants will be assigned into 6 groups with approximately equal member count
  • Each group will be responsible for presenting one topic.
  • One or two presenters.
  • The presentation should be short (5 min), include 3 main slides plus a Thanks/Contribution slide, listing the contribution of each member of the group and sources used.
  • As an option, the presentation may be recorded in advance and shared as a video on any popular video platforms.
  • The group will nominate presenters and the presentation topic ahead of time.

Types of presentations

Paper

  • 3 slides summarizing background and what was accomplished (results, methods)
  • anything that seems particularly interesting to you, open questions
Software tool
  • Motivation - what problems it solves
  • screenshots or demo

Journal Club Candidate papers and topics

For each module see possible topics below.

Module 1 - Linux for bioinformatics

You can choose one of the following topics:

  • Text data manipulation via built-in tools like awk and sed
  • Analyzing data without leaving the terminal. You can choose to cover one or more tools like:
    • qsv, xan, csvkit – CSV manipulation
    • jq/jaq - JSON manipulation
    • Biopieces
    • DuckDB – This is a versatile all-purpose database, but we can focus on its command-line CSV data processing capabilities
    • Or anything else you may find interesting and fitting this topic
  • Tools for maximizing productivity of command-line usage. Advanced customization and automation.
    • Tmux/screen
    • Other shells that could be useful for data analysis: Fish, xonsh, nushell
    • Tools of your choice

Module 2 - NGS

Choose any of the following topics for your presentation. You can take any of suggested papers / tools as a basis, but feel free to add other papers or tools that you find relevant.
Topics:

  • Alignment The foundation of everything in bioinformatics
    • Short-read alignment
      • BWA paper: Li H. (2013) Aligning sequence reads, clone sequences and assembly contigs with BWA-MEM.
      • Minimap2 aligner
      • Benchmark papers - comparison of different tools
    • Global alignment. Some recent papers:
      • anchorWave
      • A* algorithm for global alignment
  • Pangenome This is a very hot and promising research topic.
    • Review papers. It is useful to read some reviews and know the motivation for use of pangenome for crop species.
    • Technical papers. There are several "versions" of pangenome:
      • Variation graphs or Pangenome graphs - for representing all variation or all structural variation.
        • Nat. Genetics 2019 Fast and accurate genomic analyses using genome graphs
        • vg toolset - see github page and vg paper
        • Superbubbles and cacti paper
      • Genes only approach, where the goal is to gather gene models across different genomes and understand gene presence-absence variations - probably the earliest pangenome approach.
        • GENESPACE paper and github
      • Multi-reference approach
  • Workflow management systems
    • Nextflow
    • Example SNP calling workflow in rice - Yong's HPC pipeline

Module 4 - Post-GWAS

  • Online databases, resources and tools
    • Gramene, other databases
    • Rice Gene Index
  • Galaxy
    • Rice Galaxy/Crop Galaxy
    • General Galaxy resources
  • SNP-Seek:
    • Mansueto, et al. Rice SNP-seek database update: new SNPs, indels, and queries.Nucl. Acids Res.(2017) 45 (D1): D1075-D1081.
      • This paper describes new features and datasets added to SNP-Seek in 2015-2017 as well as software and database updates.
    • Alexandrov, et al. SNP-Seek database of SNPs derived from 3000 rice genomes.Nucl. Acids Res.(2015). 43(D1):D1023-D1027
      • This is the first publication introducing SNP-Seek database, data generation for Nipponbare reference,and common use cases.
    • Mansueto, et al. SNP-Seek II: A resource for allele mining and analysis of big genomic data in Oryza sativa.Current Plant Biology(2016). 7-8:16-25
      • This paper contains details on variant calling for 5 references, integration of additional genomic data, web interface, database schema, use cases, web services API.
