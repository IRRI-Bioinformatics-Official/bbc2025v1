Additional instructions specific to a software:

  1. Plink - https://www.cog-genomics.org/plink/

  • Download the binary distribution for your system from the "Binaries" table, "Stable build" column. Then just unzip the folder and move to the training directory
    Direct link as of 2024-06-06:
    Linux binary to use in WSL2
    Windows binary to use in cmd.exe / PowerShell
    MacOS users need     Mac binary

    Note - PLINK is not a regular app to be installed into Program Files. It is just a command-line executable. You only need to download it and make sure you have privileges to execute it (i.e. admin privileges). We will show an example of how to use it. To run it, you need to have a terminal-like application, on Windows it is Powershell.  
    To check installation, open Powershell, navigate to the folder containing plink.exe file, and type “    .\plink.exe --help“
    (Alternatively, use “cmd” )
  1. From the plink folder, go to the address bar and highlight the address.
  2. Type “cmd” on the address bar.
  3. Type “.\plink.exe --help”
  • Video how to install on Windows: https://drive.google.com/file/d/1Vfx4uKlNHH3wQO6ei4G-uD6v5lyW6ME1/view?usp=sharing

  1. R - https://www.r-project.org/ 

  • Mirror: https://cran.csie.ntu.edu.tw/ 
  1. To install “source-only” packages on Windows one may need
    Rtools: https://cran.r-project.org/bin/windows/Rtools/ 
    If your version of R is R-4.2.xx, install Rtools42.
    If your version of R is R-4.3.xx, install Rtools43.

  1. R packages on R-studio:

  • install.packages("qqman")
  • install.packages("ggplot2")
  • install.packages(c("dplyr", "glue", "reshape2"))
  • install.packages(c("readr", "vroom", "openxlsx"))
  • install.packages("BEDMatrix")
  • install.packages("BiocManager") # to install packages from Bioconductor
  • BiocManager::install("snpStats")
  • install.packages(c("shinydashboard", "DT", "bslib", "rmarkdown"))

  1. Optional software
  • Admixture - http://dalexander.github.io/admixture/download.htm
  • GEMMA - https://github.com/genetics-statistics/GEMMA/releases 
  • SnpEff and SnpSift - https://pcingola.github.io/SnpEff/

Note that PLINK can be installed on Ubuntu using apt:
$ sudo apt install plink1.9
but do NOT do "install plink" without a version, as it will install a different application (network utility) with the same name.

R can be installed as

$ sudo apt install r-base

Post-installation check

  1. Java

  1. Check version of Java by opening a Powershell or Terminal and issuing command:
  2. java -version

  1. TASSEL

  1. Open TASSEL, open the tutorial dataset packaged within the Tassel folder.

  1. RStudio

  1. Open RStudio, and run “version” command  in the console
  1. Install the package “qqman” using the menu.

  1. Powershell/Terminal

  1. Open Powershell (MS Windows) or Terminal (Mac OS) and note the current working directory by typing  pwd (then Enter)

  1. WSL

  1. If you are on Windows:
  2. Check WSL version
  3. wsl --version
  4. wsl --list
